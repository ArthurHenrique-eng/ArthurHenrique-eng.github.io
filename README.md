# ArthurHenrique-eng.github.io

Portfólio web de Arthur Henrique de Paula Barbosa, estudante de Desenvolvimento de Sistemas.

O site foi criado com HTML, CSS e JavaScript puro para funcionar diretamente no GitHub Pages:

- Início com apresentação profissional

- Projetos em destaque
- Experiência e competências
- Formação, certificados e contato

## Estrutura

```text
.
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── assets/
    └── img/
```

## Como trocar imagens

No arquivo `index.html`, procure pelos comentários `Troque por uma imagem do projeto`.
Depois, adicione suas imagens em `assets/img/` e atualize o caminho no HTML.

Exemplo:

```html
<img src="assets/img/lumora-app.jpg" alt="Tela do Lumora App">
```
